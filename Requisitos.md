# Ajeita Ai - API

## Requisitos Funcionais

- [ ] O usuário pode se cadastrar
- [ ] O prestador pode se cadastrar
- [ ] O usuário pode fazer login
- [ ] O prestador pode fazer login
- [ ] O usuário pode buscar prestadores
- [ ] O prestador pode receber solicitações de serviço
- [ ] O usuário pode solicitar serviços e enviar fotos do local
- [ ] O usuário pode avaliar prestadores

## Requisitos Não Funcionais

- [ ] O sistema deve ser rápido
- [ ] O sistema deve ser seguro
- [ ] A senha do usuário deve ser criptografada
- [ ] A senha do prestador deve ser criptografada
- [ ] Os dados devem ser persistidos em um banco de dados PostgreSQL

## Regras de Negócio

- [ ] O usuário deve ser maior de 18 anos
- [ ] O prestador deve ser maior de 18 anos
- [ ] O prestador pode ter uma descrição
- [ ] O prestador pode ter uma foto
- [ ] O prestador pode ter uma avaliação
- [ ] Não deve ser possível cadastrar um usuário com um e-mail já cadastrado
- [ ] Não deve ser possível cadastrar um prestador com um e-mail já cadastrado
